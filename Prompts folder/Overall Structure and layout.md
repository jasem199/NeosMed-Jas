NeosMed is a mobile-first layout. 
The entire interface is constrained to a mobile viewport style, centered on desktop screens with a max width of 600px. the background is a neutral muted color. All design decisions follow **NeosMed Design.md**.

Responsive Behavior:

- On mobile (< 430px): Full width layout
- On tablet and desktop (> 430px): max 600px wide, rest of screen filled with a muted background or subtle NeosMed branded pattern
- Font sizes never scale beyond mobile sizes even on larger screens — maintains app feel

### Accessibility

- Minimum touch target size: 44x44px on all interactive elements
- Font size base: 16px, no text below 10px. 12px could be used.
